## âœ¨ Features

### For Teachers (Maestros)
- **Class Management**: Create, edit, and manage multiple classes with unique access codes.
- **Assignment Builder**: Create complex assignments with various question types:
  - Multiple Choice & True/False
  - Fill in the Blanks
  - Word Scramble & Sentence Builder
  - Error Correction & Translation
  - Categorization & Matching
  - Short Answer & Essays
- **Interactive Gradebook**: Real-time grading, manual score overrides, and retake management.
- **Smart Dashboard**: Scoped data view focusing on recent submissions, pending tasks, and class performance.
- **Student Request System**: Teachers can send invitations to existing students to join their classes.
- **Admin Dashboard**: Specialized role-based access for the platform administrator.

### For Students (Estudiantes)
- **My Classes**: Easy enrollment and class management.
- **Interactive Learning**: Complete assignments with real-time feedback and engaging UI.
- **Practice Mode**: Ungraded practice sessions using past assignment questions (Flashcards, Quizzes, Exercises).
- **Achievements & Leaderboards**: Track progress through XP, levels, and competitive rankings.
- **Real-time Notifications**: Direct communication with teachers and class announcements.

## ğŸš€ Getting Started

### Prerequisites
- Node.js (Latest)
- Firebase Account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ferny121991/EnglishMaster-Pro.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Setup Firebase:
   - Create a project in [Firebase Console](https://console.firebase.google.com/).
   - Add your Firebase configuration to `src/lib/firebase.js`.
4. Run development server:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```
