# Portfolio Plan: fernelydev

## Overview
A professional, minimalist portfolio for a Full-Stack Developer with a premium dark mode aesthetic and neon blue accents.

## Tech Stack
- **Framework:** React
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (for smooth transitions)
- **Icons:** Lucide React

## Design System
- **Theme:** Dark Mode
- **Primary Color:** #000000 (Background)
- **Accent Color:** #00D1FF (Neon Blue)
- **Typography:** Inter or Outfit (Google Fonts)
- **Style:** Minimalist, glassmorphism, high contrast.

## Structure & Sections

### 1. Hero Section
- **Content:** Big Typography with "fernelydev"
- **Tagline:** Full-Stack Developer
- **Visuals:** Subtle animated background or a 3D element (inspired by Blender skills).
- **CTA:** "View Work" or "Contact Me".

### 2. Projects Grid
- **Layout:** Responsive grid (2 or 3 columns).
- **Cards:** Glassmorphism cards with project image, title, short description, and tech stack tags.
- **Interactions:** Hover effects with neon blue borders/glow.

### 3. Skills Section
- **Items:** HTML, CSS, JS, React, Flutter, Blender.
- **Visuals:** Icon-based display with progress indicators or simple clean badges.

### 4. Contact Form
- **Form Fields:** Name, Email, Message.
- **Style:** Minimalist inputs with neon blue focus states.
- **Submit Button:** Glowing neon blue button.

## Implementation Steps
1. Initialize Vite + React + Tailwind project.
2. Setup layout and global styles (Dark Mode).
3. Build Hero component.
4. Build Projects Grid.
5. Build Skills section.
6. Build Contact form.
7. Add global animations and refined polish.
